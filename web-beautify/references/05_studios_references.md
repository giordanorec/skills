# Studios de referência — quem fazer benchmark por estilo

> Quando o usuário pede "estilo X studio", aqui está o mapa real (2026, baseado em Awwwards rankings).

## Top studios por awards (Awwwards SOTY / SOTD / SOTM)

### Locomotive (Montreal)
- **Stats**: 131 HM, 91 SOTD, 4 SOTM, 1 SOTY
- **Vibe**: striking scroll effects, visual storytelling, design-led
- **Stack típico**: Lenis (eles criaram o "Locomotive Scroll", precursor), GSAP, custom WebGL
- **Use como ref quando**: scroll-driven narratives, brand sites premium
- **Site**: locomotive.ca

### Immersive Garden (França)
- **Stats**: 89 HM, 68 SOTD, 13 SOTM, 3 SOTY
- **Vibe**: top Three.js agency, immersive WebGL, generative art
- **Stack típico**: Three.js + GSAP + custom shaders
- **Use como ref quando**: hero WebGL, generative, brand-as-experience
- **Site**: immersive-g.com

### Active Theory (LA / Global)
- **Stats**: 76 HM, 57 SOTD, 10 SOTM, 3 SOTY
- **Vibe**: fully immersive WebGL, mid-large productions, tech-driven
- **Stack típico**: Three.js heavy, custom engines, audio integration
- **Use como ref quando**: cliente quer "experience" não "site"
- **Site**: activetheory.net

### Resn (NZ / Global)
- **Stats**: 76 HM, 61 SOTD, 11 SOTM, 2 SOTY
- **Vibe**: interactive design + audio integration, creative direction forte
- **Stack típico**: Three.js + Howler.js + custom interactions
- **Use como ref quando**: experiência audiovisual, brand-as-art
- **Site**: resn.co.nz

### Lusion (Bristol UK)
- **Stats**: bem ranqueado, foco 3D + motion
- **Vibe**: 3D + interactive + motion design
- **Stack típico**: Three.js + R3F + GSAP
- **Site**: lusion.co

### OFF+BRAND
- **Vibe**: Lando Norris site (Awwwards SOTY 2025)
- **Stack típico**: hero photography + scroll-driven 3D
- **Use como ref quando**: personality-driven sites (celebrity, athlete, brand persona)

## Designers solo / portfolios canônicos

### Bruno Simon
- **Site**: bruno-simon.com
- **Awards**: Awwwards SOTM Jan 2026 + history of awards
- **Vibe**: portfolio como mundo 3D explorável (carrinho controlável)
- **Use como ref quando**: portfolio quer ser game/playground
- **Stack**: Three.js + custom physics

### Sebastien Lempens
- **Vibe**: WebGL portfolios + tutoriais
- **Stack**: Three.js + GSAP

### Pierre Garner
- **Vibe**: motion design refinado, French school
- **Stack**: GSAP + Lenis + Vue/Nuxt

## Não-studios mas referência forte

### Apple
- Não é "studio" mas é a referência number 1 de "marketing as museum gallery"
- Stack: SF Pro + Tailored shaders + Apple-specific motion

### Stripe
- "Editorial fintech" — referência do que SaaS premium se parece
- Mesh gradient hero é trademark deles
- Sohne weight 300 + indigo `#533afd`

### Linear
- Referência de "interface calma" + dev tool aesthetic
- Near-black canvas + Inter Variable 510/590

### Vercel (Geist)
- Referência B&W high-contrast + dev-first
- Geist font + mesh gradient nos heros

### Awwwards
- Site of the Year 2025: Lando Norris (OFF+BRAND)
- Site of the Month Feb 2026: Renaissance Edition (generative + Renaissance fusion)
- Site of the Month Jan 2026: Bruno Simon portfolio
- Developer Site of the Year 2025: Messenger
- E-commerce of the Year 2025: Scout Motors

## Heurística pra escolher referência por contexto

```
projeto = portfolio criativo                → Bruno Simon, Lusion, Resn
projeto = brand campaign / launch site      → Active Theory, Locomotive, OFF+BRAND
projeto = SaaS B2B                          → Stripe, Linear, Vercel
projeto = e-commerce / product              → Apple, Nike, Airbnb
projeto = dev tool                          → Vercel, Linear, Cursor, Warp
projeto = consumer app marketing            → Spotify, Notion, Cal.com
projeto = editorial / publication           → Stripe, NYT, The Verge
projeto = enterprise                        → IBM, HashiCorp, Atlassian
projeto = artistic / experimental           → Bruno Simon, Awwwards SOTM
```
